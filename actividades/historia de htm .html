Historia del HTML

Origen y primeras versiones

HTML (HyperText Markup Language) fue desarrollado por Tim Berners-Lee en 1991 como un lenguaje para compartir documentos científicos en la World Wide Web.

HTML 2.0 (1995)

Fue la primera versión estandarizada, incluyendo soporte para formularios y tablas.

HTML 3.2 (1997)

Introdujo soporte para scripts y tablas más avanzadas, permitiendo una mayor flexibilidad en el diseño web.

HTML 4.01 (1999)

Separó la presentación del contenido mediante CSS, mejorando la accesibilidad y la estructura del código.

XHTML 1.0 (2000)

Basado en XML, promovió una sintaxis más estricta y estructurada.

HTML5 (2014)

Incorporó soporte para multimedia sin necesidad de plugins, nuevas etiquetas semánticas y APIs avanzadas.
