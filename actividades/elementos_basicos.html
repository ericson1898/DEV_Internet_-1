Elementos basicos para HTML

(canvas)que crea un espacio rectangular en blanco en el que los diseñadores web pueden dibujar usando JavaScript.

(svg) que es un contenedor para gráficos vectoriales escalables (SVG).

(header)que crea un encabezado en la parte superior de la página.

(footer) que crea un pie de página en la parte inferior de la página.

(article) que crea un área para contenido independiente.

(section)que define secciones y subsecciones como capítulos o encabezados y pies de página cuando son necesarios más de uno.

(nav)que crea un menú de navegación.

(body) marca el inicio y el final del contenido principal de la página web.

(img) se utiliza para insertar imágenes en una página web.

(a) se utiliza para crear enlaces a otras páginas web o recursos.

(div) se utiliza para agrupar otros elementos HTML en una sección lógica y separada de la página.

(span) se utiliza para aplicar estilos específicos a un fragmento de texto dentro de una página web.
